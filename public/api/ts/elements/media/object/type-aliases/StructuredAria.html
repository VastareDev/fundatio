Type Alias: StructuredAria ​

StructuredAria = object

Defined in: elements/media/object/index.ts:47

Structured ARIA input supported by Sol element factories.

Remarks ​

This is intentionally a small, typed subset that covers common cases and prevents typo-based ARIA bugs.

It is mapped into GlobalAttrs.aria for application by dom.ts.
