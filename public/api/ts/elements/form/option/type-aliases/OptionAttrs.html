Type Alias: OptionAttrs ​

OptionAttrs = Omit<GlobalAttrs, "aria"> & object

Defined in: elements/form/option/index.ts:91

Attribute bag for option creation/enhancement.

Type Declaration ​

aria? ​

optional aria: StructuredAria

Structured ARIA fields mapped into aria-* attributes.

disabled? ​

optional disabled: boolean

Whether the option is disabled.

label? ​

optional label: string

A label for the option.

Remarks ​

When present, this may be used by user agents as the option's label instead of the text content in certain contexts.

selected? ​

optional selected: boolean

Whether the option is selected by default.

Remarks ​

For <select>, selected commonly indicates the default selected option on initial load. If multiple options are marked selected, browser behavior varies; prefer a single default selection unless using a multi-select.

value? ​

optional value: string

The option value.

Remarks ​

If omitted, user agents commonly derive a value from the option's text.

Remarks ​

Options accept standard HTML global attributes plus a small set of option-specific content attributes.

Fundatio also supports a structured ARIA input for common ARIA fields, which is mapped into GlobalAttrs.aria without changing Fundatio's core DOM helpers.
