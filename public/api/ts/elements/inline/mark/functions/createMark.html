Function: createMark() ​

createMark(text?, attrs?): HTMLElement

Defined in: elements/inline/mark/index.ts:140

Create a mark element with optional text content and global attributes.

Parameters ​

text? ​

string

Optional text content for the mark element.

attrs? ​

MarkAttrs

Optional attributes to apply.

Returns ​

HTMLElement

The created <mark> element.

Remarks ​

  • Text content is assigned via textContent (never innerHTML).
  • Global attributes are applied via Fundatio's shared DOM helper, including security guards that block inline event handler attributes (e.g. onclick) and raw style attribute strings.

Example ​

ts
import { createMark } from "@Vastare/Fundatio/elements/inline/mark";

// Highlight a search match:
document.body.appendChild(createMark("kitchen", { className: "match" }));
