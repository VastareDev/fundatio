Function: createOutput() ​

createOutput(text?, attrs?): HTMLOutputElement

Defined in: elements/form/output/index.ts:180

Create an output element with optional text content and attributes.

Parameters ​

text? ​

string

Optional text content for the output.

attrs? ​

OutputAttrs

Optional attributes to apply.

Returns ​

HTMLOutputElement

The created <output> element.

Remarks ​

  • Text content is assigned via textContent (never innerHTML).
  • Global attributes are applied via Fundatio's shared DOM helper, including security guards that block inline event handler attributes (e.g. onclick) and raw style attribute strings.

Example ​

ts
import { createOutput } from "@Vastare/Fundatio/elements/form/output";

const out = createOutput("0", { name: "result", for: "a b" });
